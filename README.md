# Ecommerce Manual Testing Project

## Project Overview
This project focuses on end-to-end manual testing of an E-commerce web application.
It validates core functionalities, UI behavior, and user experience.

## Application Name
ShopEasy – E-commerce Web Application

## Modules Tested
- User Registration
- Login & Logout
- Product Search
- Product Details
- Add to Cart
- Checkout
- Payment (Dummy)
- Order Confirmation

## Testing Types
- Manual Testing
- Functional Testing
- UI Testing
- Smoke Testing
- Regression Testing

## Test Artifacts
- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- Test Summary Report
